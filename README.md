# Vue Boilerplate

Personal Vue-CLI boilerplate with my preferred linting settings and favorite packages included.

## Packages

 - axios
 - vue-router
 - vuex
 - vue-skip-to

## Start a New Project

```
git clone https://github.com/Abbe98/vue-boilerplate.git
cd vue-boilerplate
rm -rf .git/
git init
```
